# 🛡️ TGProxy: Ваш надежный мост к стабильному Telegram

**TGProxy** — это профессиональное решение для обеспечения бесперебойного доступа к Telegram. Мы создали этот инструмент для тех, кому важна стабильность связи, высокая скорость загрузки медиафайлов и полная приватность данных.

---

### ✨ Почему выбирают TGProxy?

* **Маскировка трафика (WSS):** Использование протокола WebSocket делает ваш трафик неотличимым от обычного посещения веб-сайтов, что позволяет эффективно обходить сетевые ограничения.
* **Индивидуальная настройка (HWID):** Система защиты гарантирует, что ваша копия программы настроена и оптимизирована индивидуально под ваше оборудование.
* **Умный автозапуск:** Программа готова к работе сразу после включения компьютера, не требуя ручного вмешательства.
* **Фоновый режим:** Приложение работает тихо, сворачиваясь в системный трей и не занимая место на панели задач.

---

### 🔍 Как это работает (Прозрачность)

Мы ценим доверие пользователей и придерживаемся принципов открытости в механике работы:

1.  **Локальный шлюз:** Программа создает безопасный туннель на вашем ПК (адрес `127.0.0.1`).
2.  **Шифрование:** Весь поток данных, проходящий через TGProxy, надежно зашифрован и защищен от перехвата извне.
3.  **Безопасная идентификация:** Система HWID считывает только технический серийный номер системного диска для генерации ключа активации. Мы не собираем и не храним личные данные пользователей.

---

### 📝 Простая инструкция по запуску

1.  **Запуск:** Откройте `TGProxy.exe`. При первом старте вы увидите окно с вашим HWID (идентификатором оборудования).
2.  **Активация:** Передайте этот идентификатор администратору для получения персонального ключа доступа.
3.  **Ключ:** Введите полученный ключ в программу. После этого она автоматически перейдет в рабочий режим.
4.  **Telegram:** В настройках Telegram выберите "Тип подключения: SOCKS5", адрес `127.0.0.1` и порт `1080`.

### 🛡️ Безопасность и антивирусы

Мы стремимся к чистоте нашего ПО, однако из-за использования механизмов защиты кода (обфускации), некоторые антивирусы могут проявлять излишнюю подозрительность.

> **Важно:** Ложные срабатывания (например, `Wacatac`) — это стандартная реакция систем защиты на зашифрованные Python-скрипты. Программа не выполняет никаких скрытых действий, кроме пересылки сетевых пакетов для работы Telegram. Вы можете смело добавлять файл в список исключений.
